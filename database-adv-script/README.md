# SQL Joins - Mastering Complex Queries

## Objective:
This project is aimed at mastering SQL joins by writing complex queries using different types of joins. The following joins are explored:

1. **INNER JOIN**
2. **LEFT JOIN**
3. **FULL OUTER JOIN**
